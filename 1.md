# Building a Top-Tier Portfolio

This is a **markdown-powered** blog! By writing native `.md` files, I can include code blocks seamlessly:

```javascript
// Welcome to the Elite Portfolio
console.log("Hello Elite Portfolio!");
async function sayHello() {
  await fetch('/hello');
}
```

It renders beautifully thanks to Javascript (`marked.js`) and Tailwind's Prose utilities.

### Why Markdown?
1. **Easy to Write**: No complicated databases needed.
2. **Developer Friendly**: Perfect for sharing code.
3. **Static & Fast**: Deploys instantly to Netlify.
